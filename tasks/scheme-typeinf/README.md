# Type Inference for MiniScheme

This task is to implement a type inference algorithm for a Scheme-like language.
The relevant research is soft typing and higher-order control flow analysis:

* Mike Fagan. 1991. *Soft Typing: An Approach to Type Checking for Dynamically Typed Languages*. Ph.D. Dissertation. Rice University. [https://repository.rice.edu/items/5fdc5ea3-79b9-4795-b6f2-60ddf944adeb](https://repository.rice.edu/items/5fdc5ea3-79b9-4795-b6f2-60ddf944adeb)

* Andrew Kevin Wright. 1995. *Practical Soft Typing*. Ph.D. Dissertation. Rice University. [https://repository.rice.edu/items/c7ef787b-32d6-4cd7-b149-d45a6fe3073e](https://repository.rice.edu/items/c7ef787b-32d6-4cd7-b149-d45a6fe3073e)

* Olin Shivers. 1991. *Control-Flow Analysis of Higher-Order Languages*. Ph.D. Dissertation. Carnegie Mellon University. Technical Report CMU-CS-91-145. [https://iiif.library.cmu.edu/file/Newell_box00102_fld07880_doc0001/Newell_box00102_fld07880_doc0001.pdf](https://iiif.library.cmu.edu/file/Newell_box00102_fld07880_doc0001/Newell_box00102_fld07880_doc0001.pdf)

* Robert Cartwright and Mike Fagan. 1991. Soft typing. In *Proceedings of the ACM SIGPLAN 1991 Conference on Programming Language Design and Implementation (PLDI ’91)*. ACM, New York, NY, 278–292. [https://doi.org/10.1145/113445.113469](https://doi.org/10.1145/113445.113469)

* Andrew K. Wright and Robert Cartwright. 1997. A practical soft type system for Scheme. *ACM Transactions on Programming Languages and Systems* 19, 1 (January 1997), 87–152. [https://doi.org/10.1145/239912.239917](https://doi.org/10.1145/239912.239917)
